Do you know what's HTML?

HTML stand for Hyper Text Markup Language. It a kind of language that is easiest way to make a website. HTML is not the only language to make websites. There's also other kinds of languages that can make a lot of things.
CSS is another coding language that can go in HTML codes. It's an important part of coding. Without it, you won't be able to have beautiful fonts, sizes, and colors. This website's background's color is also made by CSS. Isn't CSS important?

Javascript is a coding language that a lot of people use. It can run programs inside <script type="text/Javascript"></script> tag. Javascript is useful and it can do a lot of things.
